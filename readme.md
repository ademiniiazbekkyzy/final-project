MarkApp
=======
MarkApp is app for processing images. Need a black-and-white version of a user-uploaded image? MarkApp will make them for you. If you need to resize the image  you need MarkApp.
___
## Install dependencies
```
pip3 install pillow
```

## Run script
```
python3 main.py
```
